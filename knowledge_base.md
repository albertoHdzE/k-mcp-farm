# ContextForge Knowledge Base

## System Overview
ContextForge is a federated MCP gateway that allows multiple agents to share tools and resources securely.

## Security Model
It uses a two-layer security model:
1. Token Scoping
2. RBAC

## Key Concepts
- **Gateway**: A connection to a remote MCP server.
- **Tool**: An executable function.
- **Resource**: Static or dynamic data context.
- **Prompt**: Pre-defined instruction templates.
